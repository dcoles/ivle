The IVLE File Browser is your way to access and edit your files stored in IVLE

Getting Started with File Browser

Navigating

The Filebrowser works very much like most other file browsers. When you open the file browser you will be faced with a series of icons representing the directories (the 'folder' icon) and files (the 'paper' icons) in your home directory. Clicking on the name of a directory will open the directory while clicking the line or icon of a directory or file will mearly select it. Multiple directories or files may be selected by clicking the checkbox in the far left column. You can also browse up to a higher level directory by clicking the required folder in the breadcrumb trail near the top of the window.

Adding Directories or Files

To create a new directory or file, open the More actions... drop down box and select either New Directory or New File under Directory actions. A pop up box will appear asking for the name of the new file. Enter the name and the file will appear in the current directory. Another way is by Uploading Files from your local computer.

Editing Files

To edit a file in the text editor, click the name of the file you wish to edit in the file browser and then click Open. This will then open the file in the Text Editor screen. When finished editing the selected file click Save button to save the file to IVLE.

Deleting Directories or Files

To delete a directory or file, open the More actions... drop down box and click Delete under File actions. A pop-up window will appear to confirm that you really wish to delete the file.
